# Interactive Circle of Fifths

An interactive React + Vite circle of fifths for exploring modal triads.

## Local Development

```bash
npm install
npm run dev
```

## GitHub Pages

This project includes a GitHub Actions workflow at `.github/workflows/deploy.yml`.
Create a GitHub repository named `interactive-circle-of-fifths`, push to the
`main` branch, then enable GitHub Pages with **Source: GitHub Actions**.

For a different repository name, update the `BASE_PATH` value in the workflow to
match the repository path, for example `/my-repo-name/`.
